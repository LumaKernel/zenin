# Introduction

# Zenin UI コンポーネント

Zeninプロジェクトで使用されるUIコンポーネントのストーリーブックです。

## 技術スタック

- **Next.js**: Reactフレームワーク
- **Mantine**: UIコンポーネントライブラリ  
- **Effect.ts**: 関数型プログラミングライブラリ
- **TypeScript**: 型安全性を提供

## コンポーネント一覧

### Components/TodoList
TODOアプリケーションのメインコンポーネント。タスクの表示、作成、完了、削除機能を提供。

**主な機能:**
- タスク一覧表示
- 新規タスク作成
- タスク完了切り替え
- タスク削除
- ローディング・エラー状態表示

### Components/ThemeToggle
アプリケーションのテーマ切り替えコンポーネント。ライト・ダーク・システムテーマに対応。

**主な機能:**
- ライトテーマ切り替え
- ダークテーマ切り替え
- システムテーマ自動切り替え

## 開発ガイドライン

### コーディング規約
- 関数型プログラミングの手法を採用
- TypeScriptのstrictモードを使用
- `interface` ではなく `type` を使用
- `as` キーワードの使用を避ける
- Effect.tsのベストプラクティスに従う

### テスト駆動開発
- テストファーストで開発
- Arrange-Act-Assertパターンを採用
- 独立性のあるテストを作成

### セキュリティ
- XSS、CSRF対策を実装
- 適切なセキュリティヘッダーを設定
- 入力値の検証を徹底

## 使用方法

各コンポーネントのストーリーを参照し、プロパティの設定や状態の確認を行ってください。
モックデータを使用したストーリーでは、実際のAPI呼び出しをシミュレートしています。
